# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Short version

Be respectful. Be patient. Assume good faith.

This is a religious community project. Disagreements on fiqh are
resolved by citing authoritative sources, not by consensus or tone.
Disagreements on code are resolved by tests passing and review.

## Reporting

Report violations to the maintainer privately (same contact as in
`SECURITY.md`). The maintainer will investigate and respond within 7 days.

## Enforcement

Violations may result in a warning, temporary ban, or permanent ban
from participation in the project at the maintainer's discretion.
